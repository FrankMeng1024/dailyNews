# Mini Program Icons

This folder should contain the following icon files (PNG format, recommended 48x48 or 64x64):

## Tab Bar Icons
- `news.png` - News tab icon (inactive)
- `news-active.png` - News tab icon (active/selected)
- `audio.png` - Audio tab icon (inactive)
- `audio-active.png` - Audio tab icon (active/selected)
- `settings.png` - Settings tab icon (inactive)
- `settings-active.png` - Settings tab icon (active/selected)

## Action Icons
- `refresh.png` - Refresh/sync icon
- `play.png` - Play button
- `pause.png` - Pause button
- `forward.png` - Skip forward icon
- `backward.png` - Skip backward icon
- `plus.png` - Add/plus icon
- `close.png` - Close/X icon
- `check.png` - Checkmark icon
- `delete.png` - Delete/trash icon
- `chevron-down.png` - Dropdown arrow
- `chevron-right.png` - Right arrow

## Theme Icons
- `sun.png` - Light mode icon
- `moon.png` - Dark mode icon
- `auto.png` - System/auto mode icon

## Recommended Resources
- SF Symbols (Apple): https://developer.apple.com/sf-symbols/
- Feather Icons: https://feathericons.com/
- Heroicons: https://heroicons.com/

You can use any icon library, just ensure:
1. Icons are PNG format
2. Size is consistent (48x48 or 64x64 recommended)
3. Use appropriate colors for active/inactive states
